## Summary: Auth-Page Project
Purpose: A Flask web application with Google OAuth authentication and a recursive design pattern generator.
A Flask web application with Google OAuth authentication and a recursive design pattern generator.

## Features

- **Google OAuth Login** - Secure authentication using Google accounts
- **User Session Management** - Displays user name, email, and current time (IST)
- **Design Pattern Generator** - Generates recursive ASCII patterns (1-100 lines)
- **Responsive UI** - Clean and simple web interface
